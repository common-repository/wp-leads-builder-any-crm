== Frequently Asked Questions ==

= 1) How can I create a form using Contact Form 7 =

To create a form using Contact Form 7:

* Enable the Contact Form 7 in CRM configuration under Form Type.
* In the CRM Forms menu, select the required fields and save.
* The shortcode with the selected fields will be created in Contact Form 7.
* Place the shortcode in a post, page, or widget.

= 2) Why am I unable to submit the form created with the Contact Form? =

To resolve submission issues with a Contact Form:

* After generating the shortcode for the Contact form, go to Contact → Contact Forms → Edit view of the shortcode.
* In the Mail menu, configure all the settings and save.
* If the mail is not configured, the form will trigger an error message stating 'Submission failed'.
